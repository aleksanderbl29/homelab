## Software
I am currently running the following services in my homelab
### Services
* Uptime Kuma
* Home Assistant
* Littlelink
* Jekyll and Chripy
* Watchtower
* [Pihole and Unbound in docker](https://github.com/aleksanderbl29/docker-pihole-unbound)
### Tools
* Cloudflare tunnels
* Github pages
